# IPL Match Winner Prediction (ML + FastAPI + Streamlit + Docker + MLflow)

An end-to-end Machine Learning project that predicts the winner of an IPL match using historical match data.  
The system includes model training, experiment tracking, API deployment, frontend visualization, containerization, and CI/CD.

---

## Project Overview

This project builds a complete ML pipeline for predicting IPL match outcomes.  
It demonstrates how a machine learning model can be trained, tracked, deployed as an API, and served through a web interface.

Key technologies used:

- Python
- XGBoost
- FastAPI
- Streamlit
- Docker
- MLflow
- GitHub Actions (CI/CD)

---

## Architecture

Dataset → Feature Engineering → Model Training → MLflow Tracking → FastAPI API → Streamlit UI → Docker Deployment

---

## Features

- Data preprocessing and feature engineering
- Machine learning model using XGBoost
- Experiment tracking using MLflow
- REST API built with FastAPI
- Interactive frontend using Streamlit
- Containerized deployment with Docker
- Automated pipeline with GitHub CI/CD
